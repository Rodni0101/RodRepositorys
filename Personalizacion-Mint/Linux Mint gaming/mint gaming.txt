Últimos Drivers Gráficos (Muy importante para el Gaming)
NVIDIA
Nota1:
590 es el último driver el día de hoy, no están listos para Ubuntu/Mint
este método instala el ultimo driver nvidia empaquetado para ubuntu/mint.
los paquetes "pinning" NO SON DRIVERS

Nota2:
existe el método estándar para instalar drivers nvidia con los repositorios
add-apt-repository ppa:graphics-drivers/ppa <--
este método instala los drivers 580.95.05
con el método que muestro instala los drivers 580.105.08
No es recomendado para el gaming usar el método add-apt-repository ppa:graphics-drivers/ppa

Eliminar drivers previos:
sudo apt purge 'nvidia*'
sudo apt autoremove
reboot

curl -fsSL https://developer.download.nvidia.com/compute/cuda/repos/ubuntu2404/x86_64/cuda-keyring_1.1-1_all.deb -o cuda-keyring.deb
sudo dpkg -i cuda-keyring.deb
sudo apt update

crear / editar
sudo nano /etc/apt/preferences.d/nvidia
agregar:
Package: *
Pin: origin developer.download.nvidia.com
Pin-Priority: 600

apt search nvidia-driver
sudo apt install nvidia-driver-580
acepta DKMS, y si pregunta por Secure Boot -> NO
